is all laptop ram the same

No, Laptop memory has had even more variance than desktop. You need to be even very specific , as there are models which physically have the same contacts ,and timing but operate at different voltages, there are four types of RAM
1. DDR ram
2. DDR2 ram
3.DDR3 ram
4.RDRAM
All four have a slightly different form factor, so you CANNOT interchange them.
